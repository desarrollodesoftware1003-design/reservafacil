# ReservaFácil

Sistema web de autenticación desarrollado en Java con Servlets y MySQL.

## Funcionalidades
- Registro de usuarios
- Inicio de sesión
- Recuperación de contraseña
- Interfaz web con HTML y CSS

## Tecnologías
- Java (Servlets)
- MySQL
- HTML
- CSS

## Cómo ejecutar el proyecto
1. Clonar el repositorio
2. Importar en NetBeans
3. Configurar base de datos MySQL (puerto 3307)
4. Ejecutar en Apache Tomcat

## Base de datos
El script está en la carpeta `/database`
